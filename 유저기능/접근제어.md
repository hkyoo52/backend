## 웹사이트 제어
* 로그인 해야 리뷰 작성 가능
* 내가 작성한 리뷰만 수정/삭제 가능

## 접근 제어 방법
* 유저가 request를 장고에 보냄
* 장고에 View 직전에 request 처리
* 여기서 decorator(함수형 뷰)나 mixin(클래스형 뷰) 사용해서 제어
